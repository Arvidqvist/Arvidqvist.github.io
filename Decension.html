Decension

Turnbased 2D roguelike in a horror setting. Developed for mobiles.

This game was developed as a schoolproject in the Unity engine. We wanted to create a game with tactical turn based movement with controls that were simple enought to be used on a touchpad.
The game is got randomly generated levels which means that all playthroughs will differ from each other

I was part of the programming team taking care of generating the levels and enemybehaviours. I was also responsible for sounddesign and all animations in game
